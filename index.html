Projects

Portfolio

My goal with each project is to challenge myself to learn and apply a new technologies. My projects here are how I've developed as a software engineer and showcase my ability to pickup new technologies quickly.

Host House

Host House gave me the opportuinty to expand my skills by integrating a google maps API
into the website. It is a website similar to AirBnB, but more specifically for booking event spots. It is built with react/redux and an express backend.

Band-Together

Band Together was my first web app which helped me develop my initial skills in JavaScript.
It runs on an express server, and utilizes AJAX request to dynamically update entries throughout the website.

Noobtube

This web application was my opportunity to become more familiar with web API's. It uses RAWG video games API to seed game data. And then uses that game data to query the youtube API for videos. I also learned how to use Material UI and integrate their components for a smoother user experience. It is built with react/redux and python/flask.

Cloudify

This web application gave me a chance to integrate AWS for image/song upload along with song streaming. It is a clone of SoundCloud that is built with react/redux and python/flask.
